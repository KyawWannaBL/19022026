‚úÖ STEP 5 ‚Äî Configure Nginx for React (SPA Mode)

Create config:

sudo nano /etc/nginx/sites-available/britium

Paste:

server {
    listen 80;
    server_name 150.95.26.174;

    root /var/www/britium-enterprise/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}

Save.

Enable it:

sudo ln -s /etc/nginx/sites-available/britium /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
‚úÖ STEP 6 ‚Äî Remove Default Nginx Page

sudo systemctl restart nginx
